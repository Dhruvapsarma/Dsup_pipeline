# Dsup pipeline
A Multi-step computational pipeline

# Overview
This project is a multi-step overview of the convergent evolution of Dsup-like proteins passing a biophysical criteria. It has been made to identify a candidate for radioprotection of cells during radiotherapy. It has identified Dsup as a candidate and then optimised it using codon optimisation for human gene expression

* NCBI protein database scanning

* Biophysical filtering

* AIUPRED disorder analysis
  
* MHC-I immunogenicity screening

* Conservational analysis

* Pre-candidate mean identity summary

* Integrated 3D structure visualization (Py3Dmol)
